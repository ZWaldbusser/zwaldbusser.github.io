Module: Reports

Reports Component

This component provides an interactive dashboard for generating and visualizing three types of reports for the Point-of-Sale (POS) system:

  1. Product Usage Report: Displays the usage of each menu item over a specified date range.
  2. X-Report: Summarizes daily transactions, showing hourly order counts.
  3. Z-Report: Provides an end-of-day summary, including hourly sales and orders placed. Additionally, it includes a reset feature to clear the report's totals.

Users can select a report type, specify a date range (and an ingredient ID for the Product Usage Report), and view the results as a line chart. The component supports dynamic data fetching and interactive visualizations using Chart.js.
